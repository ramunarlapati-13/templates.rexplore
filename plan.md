# REXPLORE TECHNOLOGIES: Master Blueprint Catalog

This implementation plan provides a **Master Prompt** for every sub-category within the Rexplore Catalog. These prompts are designed to generate premium, high-fidelity React + Tailwind components that follow the Rexplore "Industrial Visionary" aesthetic.

**Master Prompt:** "Build a high-end Corporate digital architecture for a global enterprise. Use a 'Glass-Industrial' aesthetic with deep slates and electric primary accents. **Master Prompt:** "Build an industrial-grade Product Catalog. Implement a sticky sidebar for faceted filtering (Category, Material, Scale). ### 01.05 Startup
